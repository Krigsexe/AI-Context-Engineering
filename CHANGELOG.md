## About Versioning

ODIN follows semantic versioning:
- **Major version (X.0.0)**: Incompatible changes to core philosophy or structure
- **Minor version (0.X.0)**: New features that maintain backward compatibility
- **Patch version (0.0.X)**: Bug fixes and minor improvements

Each version represents hundreds of hours of testing and refinement, driven by real-world usage and community feedback.
