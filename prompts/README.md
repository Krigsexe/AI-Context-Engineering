# 🤖 Prompts ODIN

Ce dossier contient les différents prompts qui définissent le comportement et les règles du framework ODIN.

## Prompt Principal

- **`ODIN.md`**: C'est le prompt principal et la version la plus récente (v5.1.0). Il est conçu pour être utilisé comme guide de référence pour l'agent IA autonome. Il est concis, structuré et focalisé sur la mission de l'assistant de codebase.

## Prompts Archivés

Le dossier `archive/` contient les versions précédentes et les prompts spécifiques à certains modèles pour des raisons historiques et de comparaison.

- **`archive/ODIN_v4.0.0.md`**: Version 4.0.0 du prompt, très détaillée.
- **`archive/ODIN_v2.7.1.md`**: Une version plus ancienne (v2.7.1), encore plus détaillée et contenant des exemples de scripts bash.
- **`archive/MISTRAL_AUTO_AI.md`**: Un prompt plus générique, conçu pour un agent de développement généraliste et moins spécifique au framework ODIN.

Cette structure permet de maintenir une version claire et unique du prompt principal tout en conservant l'historique des versions pour référence.
